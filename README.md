## MDM9200 Script
Scipt that allow you to connect Qualcomm MDM9200 Based 3G/LTE USB Modem to your Router with OpenWRT Firmware

## Test Device
### Router
* Model    : TP-Link TL-MR3040 v2
* Flash/RAM: 4/32 MB
* Firmware : LEDE Reboot 17.01.5
* Kernel   : 4.4.140

### USB Modem
Model : Advance BT-100
